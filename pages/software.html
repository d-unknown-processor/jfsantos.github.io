Software

Developed by me

Some software I developed and distribute freely. Unless noted, all the software distributed here is under a BSD-style license.

  • AuditoryFilterbanks: cochlear and modulation filterbanks from the Auditory Toolbox translated to C++. Includes MEX wrappers.

  • Videotracker: An object tracker based on SURF features, implemented in Python and OpenCV.

  • Praat shimmer/jitter batch extractor: Praat script for extracting shimmer and jitter from a list of audio files.

Other software and resources

Below are some pointers to software and resources I found useful in my research (and did not have to write because they already existed!).

  • Praat book (Speech signal processing with Praat): http://www.fon.hum.uva.nl/david/sspbook/sspbook.pdf

  • Some very useful Praat scripts: http://www.helsinki.fi/~lennes/praat-scripts/

  • SPTK (Speech Signal Processing Toolkit): a bunch of UNIX-style command line tools for speech processing http://sp-tk.sourceforge.net/

  • libassp (Advanced Speech Signal Processor): a C library (with Tcl wrappers) for extracting features from speech signals. http://libassp.sourceforge.net/

  • WaveSurfer: http://www.speech.kth.se/wavesurfer/

  • A list of MATLAB toolboxes is also available at the MuSAE Lab website.

  • Sonic Visualizer: http://sonicvisualiser.org/
